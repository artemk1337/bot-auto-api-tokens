# Пример документации поддержки

Добавьте сюда правила поддержки, описание продукта, частые вопросы и ответы.

Например:

- если пользователь спрашивает про API-ключ, уточните его Telegram ID;
- не обещайте действия, которых нет в документации;
- если ответа в документации нет, честно скажите, что нужна ручная проверка.
